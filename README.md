# hello-py

这是一个用于学习 Python 基本数据结构与函数式编程的示例工程。

## 项目结构

- base/
  - dict.py — 字典示例
  - list.py — 列表示例
  - set.py — 集合示例
  - str.py — 字符串示例
  - tuple.py — 元组示例
- functional/
  - decorator.py — 装饰器示例
  - filter.py — `filter` 使用示例
  - map_reduce.py — `map` 与 `reduce` 示例
  - sorted.py — `sorted` 用法示例

## 快速开始

安装要求：本仓库使用纯 Python，无额外第三方依赖。确保已安装 Python 3.x。

运行示例：

```bash
python -u functional/decorator.py
```

或者运行其他模块：

```bash
python -u base/list.py
```
